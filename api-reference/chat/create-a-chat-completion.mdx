Use provider-prefixed model names (`openai/gpt-4o-mini`,
`anthropic/claude-sonnet-4.6`, `google/gemini-2.5-flash`), plain
bare-name aliases when available, or named routers
(`clearmaas/auto`).

## OpenAPI

````yaml /openapi.yaml post /chat/completions

        - role

              items:

          type: object

            name:
              type: string

              type: string
            parameters:
              type: object

            type:

          properties:
            type:
              type: string

              type: object
````
